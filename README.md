## Overview

A MATLAB-based audio signal analysis tool that compares test audio against ideal reference samples to identify distinctive words. Uses signal processing techniques to match and classify spoken words.

## Features

- **Word Classification** — Compares test audio against ideal reference signals
- **Multi-Word Support** — Handles multiple reference words for matching
- **Visual Analysis** — Generates signal plots for comparison
- **Easy Setup** — Just update file paths and run

## Quick Start

1. Open `Assignment1.m` from the `1. MATLAB Code` folder
2. Set audio file paths in the code:
   - **Line 6** — path to `tiin` audio file
   - **Line 7** — path to `chaar` audio file
   - **Line 10** — path to `aa` audio file
   - **Line 11** — path to `ii` audio file
   - **Line 14** — path to any test audio file
3. Run the code (F5 or click Run)
